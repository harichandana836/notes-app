# 📝 Notes App
A simple and modern Notes App built using HTML, CSS, and JavaScript.  
This application allows users to create, edit, delete, and manage notes with persistent storage using Local Storage.

## 🚀 Live Demo
👉 https://harichandana836.github.io/notes-app

## ✨ Features
- Add new notes
- Edit existing notes
- Delete notes
- Clear all notes
- Persistent storage using Local Storage
- Multi-line note support
- Responsive design
- Clean modern UI

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (DOM Manipulation)
- Local Storage API
- GitHub Pages (Deployment)
